招聘岗位

技术类
Java研发工程师

月薪：12k~22K
岗位职责：
1. 根据产品需求编写软件设计方案、系统架构和数据库设计；
2. 编写代码实现服务端功能；
3. 与产品经理、设计师和客户端开发团队协作完成开发任务。
任职资格：
1. 本科及以上学历，计算机相关专业；
2. 3年及以上Java项目开发经验，有互联网产品开发经验优先；
3. java基础扎实，规范的编码，了解常用设计模式；
4. 熟悉Mysql，web产品构架，了解linux下常规操作。

运维工程师

月薪：10k~20K
任职资格：
1. 本科及以上学历，计算机相关专业；
2. 3年及以上服务端运维经验；
3. 至少熟练掌握一种主流的开发语言：Java、PHP、Perl、Python、Unix Shell ... 
4. 熟悉Linux (CentOS)系统及各种运维工作中用到的命令；
5. 熟悉MySQL、Nginx、Tomcat、Redis等常用环境的配置方法，并了解相关调优方法；
6. 学习能力强，沟通协作意识良好。
工作职责
1.  维护公司线上业务系统稳定运行，收集和分析系统性能指标，定期对提高系统 可用性提出建议；
2.  通过编写脚本工具，将维护工作自动化；
3.  处理系统异常故障，保障7*24小时业务连续性；制定备份策略，并定验证；
4. 上线发布业务系统代码,并进行压力测试；
5. 开发和维护监控报警系统、自动构建工具等运维工具。

测试工程师

月薪：6k~10K
岗位描述：
1. 根据产品需求进行功能及部分性能测试；
2. 设计并编写测试用例，根据测试用例执行相关阶段的测试工作；
3. 对发现的问题及时跟进，积极与开发人员沟通并推进问题解决；
4. 根据测试结果编写测试报告，并能提出对产品有效的改进方案。
5.有良好的沟通、表达、项目分析能力，设计能力、团队合作精神。
任职资格：
1. 能完成测试用例编写、执行；
2. 接受能力强，有较好的需求理解能力和良好的沟通能力；
3. 对测试技术感兴趣，愿意学习，做事积极主动；
4. 工作细致认真，有耐心，能够胜任重复性工作；
5. 计算机软件相关专业本科以上者优先；
6. 有一个项目以上的测试工作经验者优先。

产品类
网站产品经理

月薪：8k~15K
岗位职责：
1. 负责网站相关产品策划；
2. 独立完成产品原型、逻辑和流程设计，并制作prd文档；
3. 配合其他部门完成界面UI设计和产品的开发和测试工作；
4. 跟踪所负责产品的核心数据，进行效果分析，持续完善和优化现有功能；
5. 分析行业及竞品产品，跟踪最新发展趋势，竞争对手调研。
岗位要求：
1. 2年以上产品经验，有电商交易、支付及服务流程设计经验者优先；
2. 具有良好的视觉审美能力，熟悉Axure、MindManager、VISIO、PPT等产品设计工具；
3. 有完整产品项目经验，从初期规划到产品落地，对产品有系统的理解；
4. 有结果导向意识，具有很强的执行能力和资源协调能力，积极乐观推动项目进度；
5. 具有行业敏感性，对互联网新产品保持关注，并能从中发现潜在的市场和发展方向，并能给出整体方案。

移动产品经理

月薪：8k~15K
岗位职责：
1. 负责移动端产品策划；
2. 独立完成产品原型、逻辑和交互流程设计，并制作prd文档；
3. 根据Android和IOS系统特性针对性优化产品和原型；
4. 配合其他部门完成界面UI设计和产品的开发和测试工作；
5. 跟踪所负责产品的核心数据，进行效果分析，持续完善和优化现有功能；
6. 分析行业及竞品产品，跟踪最新发展趋势，竞争对手调研。
岗位要求：
1. 2年以上移动产品经验；
2. 智能机深度使用者，同时熟悉Android和iOS系统，习惯把玩各种各样的APP；
3. 熟悉Axure、Justinmind、MindManager、VISIO、PPT等产品设计工具；
4. 有完整产品项目经验，从初期规划到产品落地，对产品有系统的理解；
5. 有结果导向意识，具有很强的执行能力和资源协调能力，积极乐观推动项目进度；
6. 实用主义、关注细节、头脑灵活、吃苦耐劳、经得起折腾者优先；
7. 具有行业敏感性，对互联网新产品保持关注，并能从中发现潜在的市场和发展方向，并能给出整体方案。

数据产品经理

月薪：8k~15K
岗位职责：
1. 负责整体产品数据运营工作；
2. 针对现有产品进行用户行为数据进行统计、分析和挖掘，定期输出数据分析报告，并基于此提出产品优化方案；
3. 设计数据统计后台，监控数据的正确性；
4. 挖掘公司平台数据产品需求，以产品的视角组织数据，为产品和市场提供基于数据的分析和建议；
5. 熟悉各大统计平台，负责网站产品SEO优化，统计分析App产品不同分发平台数据；
分析行业及竞品产品，跟踪最新发展趋势，竞争对手调研。
岗位要求：
1. 3年以上数据产品经验，有文化娱乐相关行业经验优先；
2. 了解大数据存储、计算的常用技术；对日志生成系统有了解；
3. 实用主义、关注细节、头脑灵活、吃苦耐劳、经得起折腾者优先；
4. 了解常见的运营指标及意义，熟悉常用的数据挖掘方法，包括分类、聚类、关联规则等；
5. 熟练掌握网站以及app的流量跟踪。
